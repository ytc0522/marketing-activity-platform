# :gift: 营销活动平台 - 基于微服务架构设计的高并发项目实践

> ` 营销活动平台` 项目是一款互联网面向C端人群营销活动类系统，目前支持抽奖、秒杀、拼团等各类营销活动。

## 项目介绍

营销活动平台是从营销平台独立出来的系统，支持运营人员或商家创建、修改活动，用户参加活动、领取活动奖励。
